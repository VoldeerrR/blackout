# BLACKOUT - A Web-Based Tactical FPS Game

## Overview

BLACKOUT is a low-poly, real-time multiplayer tactical first-person shooter (FPS) game designed for web browsers. Inspired by the combat mechanics of popular games like Valorant and CS2, BLACKOUT emphasizes skill-based gunplay, smooth movement, and competitive 5v5 matches.

## Features

- **Real-Time Multiplayer**: Engage in fast-paced matches with players from around the world.
- **Competitive Gameplay**: Participate in a bomb defusal mode where teams strategize to plant or defuse bombs.
- **Customizable Characters**: Create and customize your character with various options for hair, face, skin tone, and outfits.
- **Dynamic Maps**: Explore unique low-poly maps designed for tactical gameplay.
- **Smooth Controls**: Experience fluid movement and shooting mechanics that reward skill and precision.
- **Crosshair Customization**: Tailor your crosshair to suit your playstyle with various options.

## Technology Stack

- **Frontend**: Built using WebGL with Three.js or Babylon.js for rendering.
- **Backend**: Node.js with Socket.io for real-time communication.
- **Database**: Firebase or Supabase for data storage.
- **Authentication**: Google OAuth, Email, and Guest login options.
- **Hosting**: Frontend hosted on GitHub Pages and backend on a free hosting service.

## Getting Started

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the client directory and install dependencies:
   ```
   cd client
   npm install
   ```

3. Start the development server:
   ```
   npm start
   ```

4. Open your browser and navigate to `http://localhost:3000` to play the game.
